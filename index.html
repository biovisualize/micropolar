Welcome to Micropolar.

Micropolar is built around a flexible polar axis implementation. All the charts are simply composed by adding geometries to this axis. This simple structure makes it easy to extend the library with new chart types.

The current chart types are:

  • Polar dot plot
  • Polar line plot
  • Polar area chart
  • Circular bar chart
  • Clock
